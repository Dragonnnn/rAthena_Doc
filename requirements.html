System Requirements

Current available documentation and guide for rAthena.


Hardware Required

Minimal

512 of Free RAM
1 GHz of Single Core CPU
300 MB of Free Disk Space
Network/Ethernet Card for multiplayer

Recommendation

1024 of Free RAM
1 GHz of Single Core CPU
1 GB of Free Disk Space
Network/Ethernet Card for multiplayer

NOTE:
For a server with multiplayers, 10 GB of Disk Space and 2 GB of RAM are the minimum recommendation.
For a server with multiplayers, fast single core of CPU is not enough, safe choice is Dual Core or more.
rAthena currently only running single task in single core.


Platform Required

This list just showing the mostly used platform/OS for rAthena.

  • CentOS 6.x
  • CentOS 7.x
  • Windows Server 2008

For offline or testing purpose, Windows XP, Windows 7, and other Linux distros are fine to be used.
