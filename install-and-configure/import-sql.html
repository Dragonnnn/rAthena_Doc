Server Interconnection Credentials

  • Open login table, edit the first entry, by default you will get free setting.
  • Try change the userid to myserver and user_pass to myserver, don't forget use MD5 for password! or just put this query UPDATE `login` SET `userid` = 'myserver', `user_pass` = MD5('myserver') WEHERE `account_id` = 1;

NOTE: Remember the username and password! Those will be used in conf/char_athena.conf, and conf/map_athena.conf.
