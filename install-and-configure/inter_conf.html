Configure Interconnection

We are importing SQL files for single server. 1 Login-Server, 1 Char-Server, and 1 Map-Server

This interconnection config means the configuration that related with servers and MySQL

Edit this file conf/inter_athena.conf, you can open it with text editor. Just change based on the marks. Some config values in this files are being used by login-server, char-server, and map-server.

  • IP: Use the IP of your MySQL is hosted. 127.0.0.1 is localhost.
  • Port: Port of port that used by MySQL, by default is 3306.
  • ID/Username: MySQL user that has access to 'Main' and 'Log' database. We don't always need root.
  • PW/Password: Password of MySQL user above. Leave it blank if no password. But, highly recommended to make MySQL user with password!
  • DB/Database: Database name that used when you import main.sql and logs.sql
// MySQL Login server
login_server_ip: 127.0.0.1
login_server_port: 3306
login_server_id: ragnarok
login_server_pw: ragnarok
login_server_db: ragnarok
login_codepage:
login_case_sensitive: no

ipban_db_ip: 127.0.0.1
ipban_db_port: 3306
ipban_db_id: ragnarok
ipban_db_pw: ragnarok
ipban_db_db: ragnarok
ipban_codepage:
