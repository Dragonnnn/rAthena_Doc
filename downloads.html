Downloading rAthena

As a note, rAthena only available in Github. Other links that named 'rathena' are confirmed as mirror that has no on-time update or maybe scam.


Direct Download

  1. Open rAthena repository on Github by clicking this link. https://github.com/rathena/rathena/
  2. Click on Releases, just choose the top link! Download will be started automatically.
  3. Upps! Skip the step above! Click the 'zip' and copy the link and input it in address bar.

NOTE: By using this method you are unable to update rAthena to lastest version.


Git Clone

  1. On command line, or if using Windows with MSysGit, right click on new or empty directory. Execute this command: git clone https://github.com/rathena/rathena.git/
  2. Just wait until it done.

You can download 3rd party client for Git UI. Example TortoiseGit, SmartGit, or Github for Windows

If you get problem with early EOF because of post buffer size, try execute this first: git config --global http.postbuffer 524288000
